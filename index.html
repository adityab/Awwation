Awwation

create Zooming presentations!

Awwation is an app for creating Prezi-like presentations in HTML5, so they can run in modern web browsers without Flash.

It is currently in alpha. Presentations can be viewed in the latest versions of Chrome, Firefox, Safari, and IE, the iOS browser, and Firefox Mobile.

Features

  • Simultaneous Zooming and Rotations
  • Runs completely in the browser
  • Embed images
  • Supports WebFonts
  • Themes collection
  • Open-Source!
